mailto-link example 1: redirect

This example shows how to transform your "mailto:" links so that they redirect to another url (usually your contact page). The mail address of the "mailto:" link will be passed as "mailto" request parameter. Options allow you to specify:

  • which url to redirect to
  • if it should open a new window or reuse the same (default: use same window)
  • the name of the parameter to use to pass the mail address (default: mailto)
  • another url that is passed to the redirect url - usually to indicate where to redirect after submission (default: current url)
  • the name of the parameter to use to pass the mail address (default: then)
